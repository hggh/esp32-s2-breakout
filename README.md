# esp32-s2-breakout
ESP32-S2-WROOM breakout



##### BOM

| number | name |
|---| ---|
| C1 | tantalum cap 220µF SMD EIA-7343-31 |
| C2 | ceramic cap 100 nF SMD 0805 |
| C3 | ceramic cap 0.1 µF SMD 0805 |
| R1,R2 | 10k SMD 0805 |
| U1 | ESP32-S2-WROOM |
| SW1 | SMD push switch  (opt.) |
| H1 | Pin Header for uploading |
| J1, J2, J3, J4 | pin header |
